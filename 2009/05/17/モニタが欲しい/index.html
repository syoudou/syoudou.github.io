モニタが欲しい

6年前に購入したメーカー製パソコンと一緒に付いてきたモニタを今も尚現役で使用中。特に何か不具合が出てきた訳ではないけど、6年も使ってきたし、そろそろ寿命かなと。モニタの輝度も昔は70～80％ぐらいに設定していたのに、最近は常にMAX100％じゃないと暗く感じてしまう。でも、一番の理由は「もっと大きい画面にしたい！」だったりします。

そんなわけで今日は液晶モニタ選びをしたいと思います。最終的には購入までは至っていないので、そう言うのが嫌な人はブラウザのバックをクリックして下さいませ。さて、液晶選びに入る前に現在の液晶のスペックを晒しておきましょう。

  • 型番：F17R1A
  • メーカー：NEC
  • 画面サイズ：17型
  • 最大解像度：1280×1024
  • 視野角：左右140°/上下120°
  • 表示色：約1619万色(擬似フルカラー)
  • 入力コネクタ：DVI-D

TNとかVAとかいう駆動方式については調べても分からなかったのですが、視野角がもの凄く狭いのでTN方式だと思います。応答速度についても記載がないので分かりませんが、当時別の会社のTNパネルが16msなので、これも多分それくらい。TNで16msって今考えるとかなり終わってます。しかも1619万色の擬似フルカラーだし・・・。でもまぁ、良い点を上げるならば特に使っていて目が疲れるとか感じたことがないことですかね。単に私が鈍感なだけかもしれませんが。

それでは、液晶選びを始めましょう。探す液晶モニタのコンセプトは「出来るだけ安くコスパの良い物を！」です。詳しい条件はこんな感じ。

  • メーカー：特に好みとかは無い。コスパさえ良ければ。
  • 画面サイズ：22～24型ワイド
  • 最大解像度：1920×1200が理想だが1920×1080でも可。それ以下でも。
  • 駆動方式：VA方式が理想だがTNの安さは魅力。
  • 応答速度：早ければ早いほど良し
  • 視野角：TNでも今使ってる奴よりは確実に広くなるから何でも良し。
  • 入力コネクタ：HDMI入力が1つは欲しい。
  • その他機能：ピボット機能があると嬉しい。

使用目的は主にインターネットで動画投稿サイトの閲覧が多め(ニコ厨です)。PCゲームはたまにSTG(東方project)とかする程度だが、HDMI端子を利用してXBOX360を繋いでみたい。絵描きとか写真加工とかはやらないので静止画を綺麗に表示したいと言う要求はない。ただし画面に向かっている時間が長いので、あんまり目に優しくないのはちょっと・・・。予算は4万円ぐらいまでを考えてます。はい、こんな感じで↓ＧＯ！

※価格は私が調べた範囲での最安値に近い値を記載しています。この値段で確実に購入できる訳ではありません。また、液晶モニタには初期不良扱いにならない、「ドット抜け」がある場合があるので、有償ですが交換補償を行っているお店で買うと安心です。

MDT243WG

メーカーサイト

  • メーカー：MITSUBISHI
  • 価格：70000円程
  • サイズ：24.1型
  • 最大解像度：1920×1200
  • 駆動方式：VA
  • 応答速度:6ms
  • 視野角：左右178°/上下178°

理想のVAパネル。使用目的・コストパフォーマンスを考えるとVA方式ならこれの一択。ただし余裕の予算オーバー。これが4万円で買えれば私はこんなに悩んでいない。

RDT231WLM

メーカーサイト

  • メーカー：MITSUBISHI
  • 価格：25000円程
  • サイズ：23型
  • 最大解像度：1920×1080
  • 駆動方式：TN
  • 応答速度:5ms
  • 視野角：左右170°/上下160°

近々発売予定のモニタ。上のと同じMITSUBISHI製だがこちらはTNパネルで23型。これで価格は三分の一。この予算で探すならTNしか無いか。

RDT231WM

メーカーサイト

  • メーカー：MITSUBISHI
  • 価格：34000円程
  • サイズ：23型
  • 最大解像度：1920×1080
  • 駆動方式：TN
  • 応答速度:5ms
  • 視野角：左右170°/上下160°
  • その他機能：超解像度技術

これも近々発売予定のモニタ。上のモニタとほとんど同じ構成だが「超解像度」とか言う技術が搭載されている。これがあると、低解像度の映像を拡大表示したときでも、鮮明な映像になるらしい。価格差9000円の価値があるかは発売されてからレポを見るまで何とも言えない。

G2411HD

メーカーサイト

  • メーカー：BenQ
  • 価格：23000円程
  • サイズ：24型
  • 最大解像度：1920×1080
  • 駆動方式：TN
  • 応答速度:2ms
  • 視野角：左右170°/上下160°

スピーカーは付いてないが、24型で2万円台前半で購入できる激安モニタ。他にも同価格帯のモニタはあるが、スピーカーを削った分だけ、このモニタの方が良いのでは？とか期待させるそんなモニタ。何がどう違うのかは素人の私には分かりません。

W2442PA-BF

メーカーサイト

  • メーカー：LG
  • 価格：22000円程
  • サイズ：24型
  • 最大解像度：1920×1080
  • 駆動方式：TN
  • 応答速度:2ms
  • 視野角：左右170°/上下160°
  • その他機能：ピボット機能

このモニタもすっごい安いよ！しかもモニタを縦で使うことが出来るピボット機能も付いてるよ！もう何か疑いたくなるような安さです。ねぇ大丈夫なの？本当にこれで大丈夫なの？

PLB2409HDS-B1

メーカーサイト

  • メーカー：IIYAMA
  • 価格：25000円程
  • サイズ：23.6型
  • 最大解像度：1920×1080
  • 駆動方式：TN
  • 応答速度:2ms
  • 視野角：左右170°/上下160°
  • その他機能：ピボット機能

上のモニタと同価格帯でピボット機能があるモニタ。少し画面サイズが小さくなって値段は少し上昇。

はい、さらっとモニタを紹介してきました。私的にはG2411HDかW2442PA-BFのどちらかにしようかなと考えてます。RDT231WMの超解像度技術も気になるのですが、一万円以上の差を実感できるかといったら、それほどでもない気がします。後は何かきっかけがあれば買うんですけどね・・。どこかで、特価品出てないかなぁ。
